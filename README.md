## プロジェクト一覧

### [DANMAKU SOULS](./danmaku-soul)

> 弾幕 × ソウルライク アクションゲーム

![タイトル画面](./danmaku-soul/images/title.png)

弾幕シューティングの「弾を避ける快感」とソウルライクの「スタミナ管理・パリィ・重厚な戦闘感」を融合させたデスクトップ専用アクションゲーム。

| 項目 | 内容 |
|---|---|
| ジャンル | 弾幕アクション / ソウルライク |
| プラットフォーム | Windows（Electron） |
| スタック | React + TypeScript + Vite + Electron + Zustand + Canvas API |

**主な特徴**
- スタミナ管理・ジャスト回避・パリィによる戦略的な立ち回り
- ボスのHPに応じた3段階フェーズ制の弾幕パターン
- ヒットストップ・画面揺れ・パーティクルによる演出

→ [詳細はこちら](./danmaku-soul/README.md)

## 開発環境

本プロジェクト群は **Claude Desktop + Serena MCP** を使ったバイブコーディングで開発されました。

| ツール | 役割 |
|---|---|
| [Claude Desktop](https://claude.ai/download) | AIコーディングアシスタント |
| [Serena](https://github.com/oraios/serena) | ファイル操作・コードベース管理MCPサーバー |
